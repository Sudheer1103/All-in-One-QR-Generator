📱 QR Genie

QR Genie is a modern and responsive QR Code Generator Web App.
It lets you create different types of QR codes instantly — from websites to Wi-Fi, contact cards, payments, events, and more.

✨ Features
🌐 Supported QR Tools

Text / URL → Share any website link or custom text

📶 Wi-Fi → Share SSID & password to let others connect instantly

👤 vCard → Digital business card with name, phone, email, company

📧 Email → Encode recipient, subject, and body into a QR

✉️ SMS → Prefilled phone number & message QR

💳 Payment → Generate QR for UPI, PayPal, or bank transfers

📅 Event → Create event QR (title, start/end time, location)

🎨 Customization

Choose QR size (small → large)

Set foreground color

Set background color

Live QR Preview before download

📥 Output

Download QR as PNG

Works across desktop, tablet, and mobile

🖌️ Design

Clean & modern UI with gradients

Sticky navbar with branding (QR Genie)

Animated buttons, hover states, and transitions

Fully responsive layout

🛠️ Tech Stack

HTML5 → structure

CSS3 → responsive modern styling

JavaScript (Vanilla) → dynamic QR generation

QR Code API / generator → encodes input into QR

📂 Project Structure
/ (root)
├── index.html        # Home page (landing)
├── qr.html           # QR Generator page
├── styles.css        # Global stylesheet
├── script.js         # Logic for tools & QR generation
└── README.md         # Documentation

▶️ Usage

Open QR Genie in your browser (or GitHub Pages deployment).

On the Home Page, click Go to Generator.

Choose a tool (Text, Wi-Fi, vCard, Email, SMS, Payment, Event).

Fill in the required fields (e.g., Wi-Fi SSID & password).

Adjust size, color, and background.

Click Generate QR → Preview updates instantly.

Click Download QR to save your QR code.

🌐 Deployment

Works seamlessly on GitHub Pages.

Default entry point → index.html (home page).

Internal navigation goes to qr.html for generator.
